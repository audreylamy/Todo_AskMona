## Init app
1/ Git clone the repository
```
https://github.com/audreylamy/Todo_AskMona.git
```

2/ Install npm (inside server et client)
```
npm install
```

3/ Launch app
```
npm start
```

4/ Go to http://localhost:3000/

## Technology stack

+ Front-end: ReactJS, Redux
+ Other: Sass, BEM
